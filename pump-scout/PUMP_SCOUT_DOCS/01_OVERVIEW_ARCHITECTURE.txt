WHAT IS PUMP SCOUT?
-------------------
Pump Scout is an automated small-cap US stock scanner that detects early-stage
volume anomalies and Wyckoff accumulation patterns — signals that precede
significant price moves ("pumps"). It runs on Railway as a full-stack web app.

VERSION: 20.0
LANGUAGE: Python 3.11 (backend), JavaScript/React (frontend)
DEPLOYMENT: Railway (cloud PaaS)

KEY DESIGN DECISIONS
---------------------
1. NO hardcoded ticker lists — symbols come from 4 dynamic sources every scan
2. Yahoo Finance only — no paid data subscriptions required (free tier)
3. Finnhub for earnings — one batch call covers ALL symbols (no per-ticker calls)
4. PostgreSQL required for persistence — SQLite fallback = data loss on redeploy
5. All scheduler jobs use America/New_York timezone — auto-handles EST/EDT
6. AI (Claude Haiku) is used for analysis commentary, NOT for signal generation
   (signals come from deterministic indicators and scoring)
7. Hard pre-filter before AI portfolio calls — AI cannot override code-level rules
8. Telegram alerts are optional — system fully works without them
